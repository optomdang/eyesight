## 1. Tổng Quan Hệ Thống

### 1.1 Mô Hình Tổng Thể

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                           HỆ THỐNG BÀI TẬP THỊ LỰC                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐  │
│  │   BÁC SĨ    │    │  HỆ THỐNG   │    │ BỆNH NHÂN  │    │   BÁO CÁO   │  │
│  │             │    │   TỰ ĐỘNG   │    │             │    │             │  │
│  │ • Tạo chế   │    │             │    │ • Thực hiện │    │ • Theo dõi  │  │
│  │   độ tập    │───▶│ • Tạo phiên │───▶│   bài tập   │───▶│   tiến độ   │  │
│  │ • Giao cho  │    │   tập hàng  │    │ • Tạm dừng  │    │ • Đánh giá  │  │
│  │   bệnh nhân │    │   ngày/tuần │    │ • Hoàn thành│    │   kết quả   │  │
│  └─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘  │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 1.2 Ai Làm Gì?

| Vai trò | Công việc | Ví dụ |
|---------|-----------|-------|
| **Bác sĩ** | Tạo chế độ tập và giao cho bệnh nhân | Tạo "2048 mắt xa, 3 lần/ngày", giao cho bệnh nhân A |
| **Hệ thống** | Tự động tạo phiên tập theo tần suất | 0h đêm tạo phiên tập cho ngày mới |
| **Bệnh nhân** | Thực hiện bài tập theo lịch | Mở web, thực hiện bài tập 2048 đủ 3 lần |
| **Bác sĩ** | Xem báo cáo và đánh giá tiến độ | Xem bệnh nhân đã tập bao nhiêu lần, đạt bao nhiêu % |

### 2.2 Giải Thích Từng Khái Niệm

#### 📦 BÀI TẬP
- **Là gì:** Loại game/bài tập thị lực
- **Ví dụ:** Game 2048, Game Trí nhớ, Bài tập Stereopsis
- **Ai tạo:** Quản trị viên hệ thống (tạo 1 lần)

#### ⚙️ CHẾ ĐỘ TẬP
- **Là gì:** Cài đặt chi tiết cho bài tập
- **Bao gồm:**
  - Tên chế độ (VD: "2048 mắt xa - Cơ bản")
  - Loại thị lực (Mắt xa / Mắt gần / Độ tương phản / Lập thể)
  - Khoảng cách nhìn màn hình (mét)
  - Thời gian tối thiểu mỗi lần thực hiện (phút)
  - Số lần phải hoàn thành mỗi phiên (VD: 3 lần/phiên)
  - Tần suất tập (Hàng ngày / Hàng tuần / Hàng tháng / Hàng quý / Hàng năm)
- **Ai tạo:** Bác sĩ

#### 📝 GIAO BÀI TẬP
- **Là gì:** Việc giao một chế độ tập cho bệnh nhân cụ thể
- **Ví dụ:** Bác sĩ giao "2048 mắt xa" cho bệnh nhân Nguyễn Văn A
- **Có thể điều chỉnh riêng:** Mức thị lực phù hợp với từng bệnh nhân
- **Ai tạo:** Bác sĩ
- **Trạng thái:** Đang hoạt động / Tạm dừng / Hoàn thành / Đã hủy

#### 👁️ MỨC THỊ LỰC KHI TẬP
- **Là gì:** Mức độ khó của bài tập dựa trên thị lực bệnh nhân
- **Có 2 cách xác định:**
  1. **Tự động:** Lấy từ kết quả khám thị lực gần nhất của bệnh nhân
  2. **Bác sĩ chỉ định:** Bác sĩ đặt mức thị lực riêng khi giao bài tập
- **Ưu tiên:** Nếu bác sĩ chỉ định → dùng mức bác sĩ đặt. Nếu không → dùng kết quả khám
- **Ảnh hưởng:**
  - Cỡ chữ hiển thị trong game
  - Độ tương phản màu sắc
  - Độ khó bài tập

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    CÁCH XÁC ĐỊNH MỨC THỊ LỰC KHI TẬP                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  Khi bệnh nhân bấm "Thực hiện" bài tập:                                     │
│                                                                             │
│     ┌──────────────────────────────────┐                                    │
│     │ Bác sĩ có chỉ định mức riêng?    │                                    │
│     └──────────────┬───────────────────┘                                    │
│                    │                                                        │
│         ┌──────────┴──────────┐                                             │
│         │                     │                                             │
│         ▼                     ▼                                             │
│     ┌───────┐            ┌───────┐                                          │
│     │  CÓ   │            │ KHÔNG │                                          │
│     └───┬───┘            └───┬───┘                                          │
│         │                    │                                              │
│         ▼                    ▼                                              │
│  ┌─────────────────┐  ┌─────────────────┐                                   │
│  │ Dùng mức bác sĩ │  │ Lấy từ kết quả  │                                   │
│  │ đã chỉ định     │  │ khám gần nhất   │                                   │
│  └─────────────────┘  └─────────────────┘                                   │
│                                                                             │
│  ─────────────────────────────────────────────────────────                  │
│                                                                             │
│  📌 VÍ DỤ:                                                                  │
│                                                                             │
│  Bệnh nhân A:                                                               │
│  • Kết quả khám gần nhất: Mắt xa = 20/25 (mức 13)                          │
│  • Bác sĩ giao bài tập 2048 mắt xa:                                         │
│    - Trường hợp 1: Không chỉ định mức → Dùng mức 13                        │
│    - Trường hợp 2: Chỉ định mức 10 → Dùng mức 10 (chữ to hơn, dễ hơn)      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

#### 📅 PHIÊN TẬP
- **Là gì:** Lịch tập của một khoảng thời gian (ngày/tuần/tháng)
- **Ví dụ:** Phiên tập ngày 04/01/2026
- **Ai tạo:** Hệ thống tự động tạo lúc 0:00 hàng ngày
- **Trạng thái:** Chưa hoàn thành / Hoàn thành

#### 🎮 LẦN THỰC HIỆN
- **Là gì:** Kết quả của MỖI LẦN bệnh nhân thực hiện bài tập
- **Ví dụ:** Lần 1 đạt 1200 điểm trong 5 phút
- **Ai tạo:** Hệ thống (khi bệnh nhân bấm "Thực hiện")
- **Hỗ trợ tạm dừng:** Có thể lưu trạng thái game để thực hiện tiếp sau
- **Trạng thái:** 
  -  Đạt (hoàn thành + đủ điều kiện)
  - 🔴 Không đạt (hoàn thành + không đủ điều kiện)

#### ⏸️ Bước 3A: Tạm Dừng

```
Bệnh nhân bấm "Tạm dừng"
    │
    ▼
Hệ thống:
    • Lưu trạng thái game (vị trí các ô số)
    • Lưu điểm hiện tại
    • Lưu thời gian đã thực hiện
    │
    ▼

    │
    ▼
Bệnh nhân có thể:
    • Quay lại thực hiện tiếp bất cứ lúc nào (trong cùng phiên)
    • Game sẽ được khôi phục đúng trạng thái đã tạm dừng
```

    │
    ├── Điểm số >= Điểm tối thiểu (nếu có)?
    │
    └── Độ chính xác >= Yêu cầu (nếu có)?
    │
    ▼
